The kCSD-class¶

Methods carrying out the kCSD method have been implemented as a Class. Those no familiar with object oriented programming can imagine the kCSD class as machine that provided some input can perform different actions. Like a washing machine given clothes and some washing powder let’s us choose a program of washing the clothes, the kCSD-class method given information about the electrode positions and the signals recorded can perform several variants of the kCSD method.
